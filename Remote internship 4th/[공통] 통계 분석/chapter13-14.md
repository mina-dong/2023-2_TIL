# chapter 13

유의확률 : 귀무가설을 지지하는 힘  
유의수준 : 통계적 가설검정에서 귀무가설의 기각 여부를 결정하는데 사용되는 기준이 되는 확률.  

귀무가설 = 배기량과 연비는 관련성이 없다  
대립가설 = 배기량과 연비는 관령성이 있다  
P값 < a : 귀무가설을 기각하고, 대신 대립가설을 채택한다  
P값 > a : 대립가설을 기각할 수 없다.   

통계 패키지 소프트웨어, spss, sas, matlab, R, python, excel, kess 등.
